Installation

Download the package appropriate to your platform, unzip and run:

  • Windows
  • Linux
  • Mac
  • SDK for use as a target platform

Windows and Linux packages includes a Java Runtime so you really can unzip and run without making any configuration changes to your Machine. On Mac OSX uDig is able to check for Java Advanced Imaging and install if needed.

About uDig VersionXXXX

This is a release candidate, prior to the creation of a stable 1.5.x series.

Internally the package structure has changed from net.refractions.udig to org.locationtech.udig allowing developers to upgrade their plugins in anticipation of the 2.0 release.

SDK Features:

  • Access to GeoTools 11.2 and ImageIO-Ext 1.1.9 libraries
  • Project plugins, extension points and packages have been renamed to org.locationtech.udig
