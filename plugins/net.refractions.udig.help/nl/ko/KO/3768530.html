연습 하기1

1. 우리지도 그리기(uDig) 설치 및 실행

윈도우에서 설치하기:

  1. "uDig1.1.exe" 를 더블클릭하세요. 
  2. "next" 버튼을 클릭하세요.
  3. 라이센스 동의서가 나타납니다. uDig는 EPL+BSD 라이센스를 따르고 있습니다. "I Agree" 를 클릭하세요.
  4. uDig를 설치할 프로그램 파일 디렉토리를 설정하고 "Next" 를 클릭하세요.
  5. Shortcut을 시작메뉴에 설치할수 있습니다. "Install" 을 클릭하세요.
  6. uDig가 설치되는 동안 잠시 기다려 주세요.
  7. uDig설치가 완료되었습니다. "Finish" 를 클릭하세요.

2. 환영 뷰


  1. 설치가 완료된 후, "시작 > 프로그램> uDig 1.1-RC14" 에서 uDig를 실행시키세요.
  2. uDig 를 처음 실행하는 경우 환영 뷰가 나타납니다.
  3. 환영 뷰의 "Getting Started" 는 온라인 documentation 과 연결되어 있습니다. 다음 장에서 온라인 doucumentation의 사용에 대해서 설명하겠습니다.
  4. 환영 뷰의 "Official Website" 는 uDig 홈페이지와 연결되어 있습니다.
  5. 환영 뷰의 오른쪽 상단의 "Workbench" 를 클릭하여 uDig의 작업공간으로 들어가세요.

3. 온라인 doucumentation과 튜토리얼

  1. "Help" 메뉴를 열고 "Help Contents" 를 선택하세요.
  2. 여러 항목별로 정리된 온라인 help 시스템에서 원하는 페이지를 자유롭게 볼 수 있습니다. 
  3. 콘텐츠 뷰는 다음과 같은 항목들로 구성되어 있습니다.
    • User Guide
    • Live User Guide - uDig 웹 싸이트에 실시간 접근합니다.
  4. 인스톨한 plug-in의 커뮤니티를 통해서 새로운 항목을 추가하여 이용할 수 있습니다.

4. 워크벤치

지도 편집 작업을 시작하기 전에 uDig에서 기본적으로 제공하는 핵심 레이아웃을 볼수 있습니다. 프로젝트 뷰, 레이어 뷰, 카탈로그 뷰, 지도 편집기는 uDig의 일반적인 부분으로 뒤에서 자세히 설명하겠습니다.

5. 웹 맵 서버에 연결하기

  1. drag and drop 을 포함하여 uDig에는 지도 데이터를 로드하는 많은 방법이 있습니다. uDig에서 웹 맵 서버를 drag하기 위해 웹 브라우저를 여세요.
  2. 웹 브라우저에서 walkthrough 1 페이지에 연결하세요.
    • http://udig.refractions.net/confluence/display/EN/Walkthrough 1 
  3. 웹 페이지의 DM Solution WMS를 클릭하여 레이어 창으로 drag하여 drop하세요.
  4. 레이어 추가 마법사가 나타나 WMS에서 보고자 하는 지도를 물어봅니다. "Elevation/Bathymetry", "Park", "Cities" 를 선택하고 "종료" 를 클릭하세요.
  5. 지도 레이어가 지도 창에 지도를 그리고 있습니다. uDig 애플리케이션의 오른쪽 하단의 프로그래스 뷰는 레이어를 그리거나 작업이 요구되는 동안 처리과정을 보여줍니다.
  6. 레이어의 랜더링이 끝나면 지도 뷰 에 전시된 레이어를 볼 수 있습니다.
  7. 다음의 도구를 스크린의 데이터에 사용할 수 있습니다.
    • 줌 툴(키보드 단축키 Z) : 클릭 또는 드래그 하세요. 마우스 왼쪽 버튼은 줌 인, 오른쪽 버튼은 줌 아웃 입니다.
    • 팬 툴(키보드 단축키 P) : 화면을 이동해서 볼수 있도록 원하는 곳 또는 방향으로 클릭 또는 드래그 하세요.
    • 확장 툴 : 화면 전체를 보고싶을때 클릭하세요.
  8. 레이어 창은 레이어가 그려진 순서를 보여줍니다. "Cities"  레이어를 선택하고 레이어 뷰 툴바의 "선택한 레이어 위로 이동" 을 클릭하세요.
  9. 순서가 바뀐것을 확인할 수 있습니다.

6. 카탈로그 뷰로부터 레이어 추가하기

WMS로 부터 레이어 추가하기:

  1. uDig의 카탈로그 뷰의 "DM Solutions' WMS Demo Server"*의 커넥션을 확장하세요. 확장된 커넥션중 *"Provincial Borders" 레이어를 오른쪽 클릭하고 "현재 지도로 추가" 를 선택하세요.
  2. 레이어 뷰에 새로운 레이어가 추가되고, 자동으로  랜더링이 시작됩니다. 랜더링이 끝나면 지도 위에 각 주의 경계가 검은색 선으로 표시됩니다.
  3. 카탈로그 뷰는 "지도 그래픽" 이라는 도구를 가지고 있습니다. 카탈로그 뷰의 "지도 그래픽 > 스케일 바" 를 선택해서 지도 위로 드래그 하세요.

7. PostGIS로 부터 레이어 추가하기

  1. 파일 메뉴에서 "새로운 작업 > 새로운 지도" 를 선택하세요.
  2. 프로젝트 뷰 안에 새로 생긴 "지도" 를 오른쪽 클릭 하고 "추가" 를 선택하세요.
  3. 데이터 소스 형식을 "PostGIS" 를 선택하고 다음을 누르세요.
  4. 다음의 연결정보를 입력하세요.
    • Host : www.refractions.net
    • Port : 5432
    • Username : demo
    • Password : demo
    • Database : demo-bc
  5. 연결정보 입력 후 "완료" 를 누르세요.
  6. "bc_hospitals" 와 "bc_manicipality" 를 선택하고 "완료" 를 누르세요.
  7. 줌 아웃 상태에서 랜더링 할수록 걸리는 시간이 줄어듭니다.
  8. 레이어 뷰의 "bc_hospitals" 를 오른쪽 클릭하고, "레이어를 줌" 을 선택하세요.
  9. "bc_hospitals" 레이어가 줌 인 되어 화면 전체에 확장되어 보입니다.
  10. "네비게이션 > 뒤로가기" 를 선택하여 이전의 상태로 돌아갈 수 있습니다.

8. 파일 시스템 환경에서 레이어 추가하기

  1. 링크를 오른쪽 클릭하고 "다른 이름으로 대상 저장" 을 누르세요. 저장할 위치를 정하고 다운로드 받으세요.
  2. 다운로드된 압축파일을 디렉토리에 풀어놓으세요.
  3. 압축이 풀린 디렉토리에서 "bc_border.shp" 파일을 지도 편집기에 드래그 하세요.

  4. 9. 카탈로그에서 직접 가져오기

  1. 카탈로그 뷰의 "가져오기" 를 누르세요.
  2. "데이터" 를 선택하고, '다음" 을 누르세요.
  3. 목록에서 "웹 지도 서버" 를 선택하고, "다음" 을 누르세요.
  4. JPL World Map Service를 "가져오기" 하기 위해 다음의 URL를 입력하세요.
    • http://wms.jpl.nasa.gov/wms.cgi?Service=WMS&Version=1.1.1&Request=GetCapabilities
  5. "완료"를 누르세요. 카탈로그 뷰에 "JPL Global Imagery Service"가 추가되어 있습니다.
  6. 카탈로그 뷰에 추가된 항목을 가져올수 있습니다. "Blue Marble, Global Modis derived image"레이어를 "지도 편집기"로 드래그 하세요.
  7. "선택된 레이어 위로 이동", " 선택된 레이어 밑으로 이동" 버튼을 사용해서 레이어의 순서를 재 조정 할수 있습니다. "Blue Marble" 레이어를 선택해서 맨 밑으로 드래그 하세요.

10. 주제 데이터

  1. uDig 카탈로그 뷰 안의 "JPL Global Imagery Service" 목록에서 "Blue Marble Next Generation, Global MODIS derived image" 를 선택하고 마우스 오른쪽 클릭 하세요. 그리고 "새로운 지도에 추가" 를 클릭하세요.
  2. 새로운 지도의 이름이 너무 길어 이름을 변경하겠습니다. 이름을 변경할 지도를 선택하고 오른쪽 클릭하여 "이름 바꾸기" 를 선택하세요.
  3. "Blue Marble" 이라고 입력하고 "확인" 을 누르세요.
  4. 레이어 메뉴에서 "추가" 를 선택하세요.
  5. 목록에서 "파일" 을 선택하고 "다음" 을 누르세요.
  6. 데이터 디렉토리에서 다음의 파일을 선택하세요.
    • countries.shp
    • clouds.jpg
  7. "열기" 를 누르면 두 레이어 모두 레더링이 되어 추가 됩니다.
  8. 레이어 뷰의 "clouds.jpg"를 맨 밑으로 드래그 하세요.
  9. countries 레이어가 약간 투명한 스타일을 가지고 있어서 아래의 WMS레이어를 볼 수 있습니다.
  10. countries 레이어를 오른쪽 클릭 하고 "스타일 변경" 을 선택하세요.
  11. "스타일 편집기" 는 3개의 페이지 목록으로 구성되어 있습니다. "심플 스타일 페이지" 를 사용해서 countries 레이어에 label을 추가하겠습니다.
    • 레이블을 체크하세요.
    • 목록에서 "CNTRY_NAME"을 선택하세요.
  12. "APPLY" 버튼을 누르세요. 지도에 레이블이 표시되는 것을 확입할 수 있습니다.
  13. "테마 스타일 페이지" 로 변환하고 다음의 옵션을 선택하세요.
    • 속성 : POP_CNTRY
    • 정규화 : SQKM
  14. 여러 색이 모여있는 팔레트에서 이용 가능한 목록들만 필터링 하여 모여줍니다.
    • "보기" 의 목록에서 "모든"을 선택하세요.
    • 부적합한 것 필터링에서 "LCD" 와 ""CRT" 버튼을 누르세요.
    • "light orange to dark red" 를 선택하세요.
  15. "확인" 버튼을 누르세요.
  16. 레이어 메뉴를 열고 "범례" 를 선택하세요.
  17. 레이어 뷰의 "마일라" 버튼을 누르세요. 선택한 레이어가 강조되어 나타납니다.
  18. "Blue Marble" 탭을 더블클릭하면 지도 편집기가 최대크기로 확대되며, 줌 툴을 이용하여 탐색할수 있습니다.
  19. "Blue Marble" 탭을 한번더 더블클릭하면 이전의 크기로 되돌아 갑니다. "창 > 퍼스펙티브 재설정" 을 사용하면 현재 지도의 퍼스펙티브를 기본값으로 재설정 할 수 있습니다.

11. 정보 보기

  1. 프로젝트 창의 "Elevation/Bathmetry" 지도를 활성화 하세요.
  2. 녹색 공원이 있는 지역으로 줌 하세요.
  3. 레이어 뷰에서 "Parks" 레이어를 선택하세요.
  4. 툴 바 에서 "정보 툴" 로 변환하고 지도의 공원을 누르세요.
  5. 정보 뷰가 활성화 되면서 공원에 대한 정보가 전시됩니다.
  6. 왼쪽의 뷰에서는 이용가능한 정보 목록이 전시되므로 원하는 정보를 선택하면 오른쪽 뷰에 해당 정보가 전시됩니다. 
  7. 모든 웹 지도 서버에서 정보를 제공하는 것은 아닙니다.

12 웹 카탈로그

  1. 새로운 지도를 만들고 "웹 뷰" 의 탭을 누르세요.
  2. "dm solution" 을 클릭하세요.
  3. 리소스 선택 화면이 전시되면 "Parks", "Provincial Border", "Roads", "Cities" 를 선택하고 완료를 누르세요.

13. Re-Projection

  1. 줌과 팬 툴을 사용하여 British Columbia(북 아메리카 서쪽)으로 이동하고, 지도 편집기 하단의 "WGS 84" 버튼을 투르세요.
  2. 환경설정 페이지가 전시되는데 현재 지도에 대한 좌표 시스템을 변환할 수 있습니다. 42304 를 입력하고 "NAD83/NRCan LCC Canada" 를 선택하세요.
  3. "Custom CRS" 탭을 선택하여 ESPG:42304에 대한 정의를 볼수 있습니다.
  4. "확인" 버튼을 누르면 새로운 좌표 시스템이 적용되어 지도가 재투영된 것을 확인할 수 있습니다.

14. 커뮤니티

  1. "도움 > 찾기, 설치" 을 클릭하세요.
  2. "설치할 새 기능에 대해 검색" 을 선택하세요.
  3. "새 원격사이트" 를 누르고 다음의 내용을 넣으세요.
    • 이름 : uDig Community Updates
    • URL : http://udig.refractions.net/update-community
  4. "uDig Community Updates" 를 체크하고 "완료"를 누르세요.
  5. 업데이트할 내용을 확인하는 동안 약간의 시간이 소요됩니다.
  6. 완료되면 업데이트할 목록을 확인할 수 있습니다.
  7. 목록에서 "Visual feature Feature 1.0.0" 을 체크하고 다음을 누르세요.
  8. 기능 라이센스 화면이 전시되면 "라이센스 계약의 항목 승인" 을 체크하고 "다음" 을 누르세요.
  9. 마지막으로 설치될 위치를 확인하고 "완료" 를 누르세요.
  10. 많은 커뮤니티가 제공하는 기능이 서명되어 있지 않습니다. 서명에 관한 경고 메시지가 뜨면 "모두 설치" 를 누르세요.
  11. 변경사항을 적용하기 위해 workbench를 재시작 합니다. "예" 를 누르세요.
  12. 카탈로그 뷰의 "지도 그래픽" 콘텐츠로 "North Arrow" 가 추가 되었습니다. 지도에 드래그 하여 넣으세요.
  13. 툴 바의 빨간 사각형을 누르거나 "North Arrow" 레이어를 누르고 지도위의 원하는 지점을 누르면  "North Arrow" 를 이동시킬 수 있습니다.
  14. 커뮤니티에서는 지도에서 부터 편집기에 이르기 까지 다양한 데이터를 제공합니다.
