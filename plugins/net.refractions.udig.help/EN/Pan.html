The pan tool is used to control the location of your Map. In order to use the pan tool you must have a map open.

  1. Open the swamps map and select the pan tool .
  2. Click on a point on the map. Do not release the mouse button.
  3. While still holding the mouse button down, move the mouse in any direction.
  4. Release the mouse button.

Using CTRL + (arrow key) can also be used to pan.
