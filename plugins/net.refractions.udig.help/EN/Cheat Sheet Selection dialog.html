Cheat Sheet Selection dialog¶

The cheat sheet selection dialog is available using Help ‣ Cheat Sheets in the menu bar. The dialog is used to review available cheat sheets (or otherwise select a cheat sheet from a file or URL) and display the result in the Cheat Sheet view.
