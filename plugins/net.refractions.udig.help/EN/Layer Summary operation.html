Layer Summary operation¶

Layer summary provides a quick rundown of known information for the selected layer. The operation is available from the Operations dialog or using the context menu when selecting a layer.

Selection¶

The operation works with the currently selected layer in the Layers view.

Result¶

A dialog box is opened providing a summary.
