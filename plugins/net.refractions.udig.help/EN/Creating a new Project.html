Creating a new Project¶

To create a new project:

  1. From the menubar, select File ‣ New ‣ Project

  2. In the New Project field, type a filename.

    (You may also use the Browse button)

  3. Click Finish when you are done.

The new Project has been created and is now in the Project Explorer
