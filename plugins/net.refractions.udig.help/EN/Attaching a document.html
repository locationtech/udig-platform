Attaching a document¶

To attach a document to a layer or feature:¶

  1. Load documents related to layer or feature
  2. Select the layer or feature’s folder or a document inside the folder
  3. Click Attach... to open the Add Attachment dialog
  4. Provide the file to be attached by doing one of the following:

  • Manually set the file’s path to File

  • Select an existing file
    • Click Browse... to open the file selection dialog
    • Select a file from the dialog
    • Click Open to set File with the selected file’s path
  • Create a new file from template
    • Click New... to open the select template dialog
    • Select a template from the list
    • Click OK to open the save file dialog
    • Select directory and enter filename
    • Click Save to set File with the new file’s path
  1. (Optional) Check Set as template to set attachment as a template
  2. Set the Label with appropriate label
  3. (Optional) Set the Description with appropriate description
  4. (Optional) Click Open to verify that set file is valid and existing
  5. Click Attach to save attachment

Sample dataset:

  • Try out australia.shp in data_1_4.zip of the Quickstart sample data.
