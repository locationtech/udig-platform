Remove Vertex Tool¶

Delete Vertex Tool – Cannot resolve external resource into attachment.¶

The delete vertex tool can select features and delete vertices from the features. To delete a vertex select a feature then click on the vertices to delete

Tip if features overlap clicking the same location
