The update manager provides functionality to search for new features/plugins for udig. To search for new features:

  1. Start the Working with the Update Manager

  2. Select Search for new features to install option.

  3. Press Next

  4. If the site you wish to search is not in the list press New Remote Site

  5. Enter the location and a name for the site

  6. Press OK

  7. Press Finish

  8. The new features will appear in a dialog and you can check the features you wish to install

  9. Agree or disagree to the license (The feature won’t be installed if the license isn’t accepted)

  10. If there are multiple udig installation select the correct installation and press Finish

  11. Press Install or Install All

  12. It is recommended you restart as apply changes doesn’t always work.

  13. The new feature is now installed. All you have to do is figure out how to use it.
