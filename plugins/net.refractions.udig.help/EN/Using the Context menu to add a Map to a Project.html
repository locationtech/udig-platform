Using the Context menu to add a Map to a ProjectÂ¶

You can use the context menu in the projects view to add a new map to a project.

  1. Right click on the recently created map
  1. Select Add....
  2. Select “Files” from the list of available wizards.
  3. Browse to your sample data directory (sample data)
  4. Open the directory and select one or more of the files available.
  5. Press Open

The shapefiles will be added to the map and will be displayed on the screen.
