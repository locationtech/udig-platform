We can use a layer feature to define the current boundary.

  1. Open the AOI view
  2. Select the AOI Layer option form the Boundary combo box
  3. Select the AOI Selection Tool
  4. In the tool options select a AOI Layer to work with
  5. On the map Single Click or Alt Click and Drag to select the features to use to define the current AOI.
