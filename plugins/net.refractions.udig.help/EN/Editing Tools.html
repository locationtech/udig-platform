Editing Tools Overview¶

This is the root page of the user documentation for the uDig Editing Tools extensions. Axios is providing under the Spatial Operations and Editing Tools project promoted by the Diputación Foral de Gipuzkoa, Departamento de Movilidad y Ordenación del Territorio.

These extensions comprise some additional tools for the geometry based edition of features.

Check the links bellow for specific user documentation for each of the available extensions.

  • Arc Tool — Creates a linestring approximating an arc given three control points (initial point, axis, and end point).
  • Parallel Tool — Creates a parallel LineString.
  • Split Tool (Axios Tools) — Splits features crossed by a line string
  • Trim Line Tool — Trims LineStrings at a point using a trimming line
