To set a hotlink attribute:

  1. Open the Documents view
  2. Select a feature using the Feature Selection tool
  3. The list of documents associated with this feature will be shown in the Documents view
  4. Select the hotlink document from the list
  5. Press the Edit... button, a dialog will be provided to input details
  6. Press the Commit button to write this change out as the hotlink is stored in the assigned attribute
