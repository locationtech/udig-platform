Icons and buttons¶

The application has a consistent set of icons and buttons used to refer to information and functionality.

  • Catalog view
  • Projects view
  • Layers view
  • Toolbar
  • Map Palette
  • Page Palette

This page offers a quick visual reference and may be used as a starting point to look up more detailed information.
