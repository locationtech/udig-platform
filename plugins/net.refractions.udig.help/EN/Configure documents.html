Configure documents¶

To open settings page:¶

  1. Use Catalog view to select the resource or select a layer from the Layers view
  2. Open the Resource page from the Data menu.
  3. Select the Document page tab

To enable resource documents:¶

  1. Open settings page
  2. Check Enable support for resource documents

To enable feature documents:¶

  1. Open settings page
  2. Check Enable support for feature documents

To enable feature hotlinks:¶

  1. Open settings page
  2. Check Enable support for hotlink on marked attributes

To set an attribute as a file hotlink:¶

  1. Enable feature hotlinks
  2. Click Add... to open the Add Hotlink Descriptor dialog
  3. Select File as the Type
  4. Select the Attribute containing the reference information
  5. Provide the appropriate Label
  6. (Optional) Provide the appropriate Description
  7. Click OK

To set an attribute as a web hotlink:¶

  1. Enable feature hotlinks
  2. Click Add... to open the Add Hotlink Descriptor dialog
  3. Select Web as the Type
  4. Select the Attribute containing the reference information
  5. Provide the appropriate Label
  6. (Optional) Provide the appropriate Description
  7. Click OK

To set an attribute as an action hotlink:¶

  1. Enable feature hotlinks
  2. Click Add... to open the Add Hotlink Descriptor dialog
  3. Select Action as the Type
  4. Select the Attribute containing the reference information
  5. Provide the appropriate Label
  6. (Optional) Provide the appropriate Description
  7. Provide the Action definition
  8. Click OK

Action definition examples:

  • https://www.google.com.au/search?q={0} - For a Google search action
  • http://au.search.yahoo.com/search?p={0} - For a Yahoo! search action
  • C:\Reports\{0}.pdf - For a file reference action

Note: Use the marker ‘{0}’ in the action definition to mark parts to be replaced with the attribute’s value.

To edit an existing hotlink definition:¶

  1. Open settings page
  2. Select hotlink from the list
  3. Click Edit... to open the Edit Hotlink Descriptor dialog
  4. Select Type
  5. Select the Attribute containing the reference information
  6. Provide the appropriate Label
  7. (Optional) Provide the appropriate Description
  8. (For Action types) Provide the Action definition
  9. Click OK

To remove an existing hotlink definition:¶

  1. Open settings page
  2. Select hotlink from the list
  3. Click Remove

Sample dataset:

  • Try out australia.shp in data_1_4.zip of the Quickstart sample data.
