To open settings page:

  1. Use Catalog view to select the resource or select a layer from the Layers view
  2. Open the Resource Properties from the Data menu.
  3. Choose the Document property page

To enable resource documents:

  1. Check Enable support for resource documents

To enable feature documents:

  1. Check Enable support for feature documents

To enable feature hotlinks:

  1. Check Enable support for hotlink on marked attributes

To mark attributes as hotlinks:

  1. List the attributes you want as hotlinks
    • Use the Add... to add an attribute as a hotlink
    • Use the Edit... to update an existing hotlink definition
    • Use the Remove to remove an attribute as a hotlink

A sample dataset is provided:

  • Try out australia.shp
  • File, Web and Action hotlinks are already defined
