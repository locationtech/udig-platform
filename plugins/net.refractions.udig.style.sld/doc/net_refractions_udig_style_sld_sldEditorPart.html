SLD Content Extension

Identifier:
net.refractions.udig.style.sld.sldContentExtension

Since:
1.0

Description:
An extension point for Style Layer Desriptor (SLD) style content.

Configuration Markup:

<!ELEMENT extension (sldEditorPart)*>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>

An extension point for adding ui components used to edit a component of an SLD style.



<!ELEMENT sldEditorPart EMPTY>

<!ATTLIST sldEditorPart

class       CDATA #REQUIRED

label       CDATA #REQUIRED

contentType CDATA #REQUIRED

>

  • class - An subclass of SLDEditorPart
  • label - A human readable label used to identify the sld extension in the ui. For example, "Line", "Polygon", etc.
  • contentType - The class of style component of interest. One of Style, FeatureTypeStyle, Rule, or Symbolizer.

API Information:
