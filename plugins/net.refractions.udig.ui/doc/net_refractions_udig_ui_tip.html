tip

Identifier:
net.refractions.udig.ui.tip

Since:
[Enter the first release in which this extension point appears.]

Description:
A tip is a usage tip that will be displayed upon startup. A single tip will be shows each startup.

Configuration Markup:

<!ELEMENT extension (tip* , configuration* , activeConfiguration?)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>

Tip extension declaration.



<!ELEMENT tip EMPTY>

<!ATTLIST tip

id   CDATA #REQUIRED

name CDATA #REQUIRED

icon CDATA #IMPLIED

>

A single tip to be displayed. The text that will be displayed is the body text of the tip element.


  • id - id of the tip
  • name - Name of the tip. This will be displayed in the title bar of the tip dialog
  • icon - Icon shown in the tip dialog

<!ELEMENT configuration (tipRef* , tipExtensionRef*)>

<!ATTLIST configuration

id   CDATA #REQUIRED

name CDATA #REQUIRED

>

configuration declares a configuration of tips that will be shown.

By default all tips are shown


  • id - ID of the configuration
  • name - Human readable and translateable name of the configuration.

<!ELEMENT activeConfiguration EMPTY>

<!ATTLIST activeConfiguration

configurationID CDATA #REQUIRED

>

  • configurationID - Defines which configuration is active.

<!ELEMENT tipRef EMPTY>

<!ATTLIST tipRef

tipID CDATA #REQUIRED

>

References a tip. This is a reference to a tip used for defining a configuration.


  • tipID -

<!ELEMENT tipExtensionRef EMPTY>

<!ATTLIST tipExtensionRef

extensionID CDATA #REQUIRED

>

References an extension of tips. This is a reference to a tip used for defining a configuration.


  • extensionID -
