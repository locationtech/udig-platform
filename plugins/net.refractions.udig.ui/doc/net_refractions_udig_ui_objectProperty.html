%objectProperty.point

Identifier:
net.refractions.udig.project.ui.objectProperty

Since:
1.1.0

Description:
This point allows a true or false property to defined for an arbitrary object. It is used by the enablesFor element of the tool extension point.

Configuration Markup:

<!ELEMENT extension (object+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>

This point allows a true or false property to defined for an arbitrary object. It is used by the enablesFor element of the tool extension point.



<!ELEMENT object (property+)>

<!ATTLIST object

targetClass CDATA #REQUIRED

>

  • targetClass - The class that the property operates on. It will be passed in as a parameter of the property's isTrue method.

<!ELEMENT property EMPTY>

<!ATTLIST property

id    CDATA #REQUIRED

class CDATA #REQUIRED

>

  • id - the id of the property value that will be used. It will be concatenated with the namespace identifier of the plugin defining the extension. For example a property "prop" in plugin "example.plugin" is referenced by "example.plugin.prop".
  • class - The PropertyValue implementation. It is not required to be an net.refractions.udig.ui.operations.AbstractPropertyValue but it is recommended. At minimum it must be a net.refractions.udig.ui.operations.PropertyValue
