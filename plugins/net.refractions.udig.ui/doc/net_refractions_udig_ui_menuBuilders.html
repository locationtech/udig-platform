Menu Builders

Identifier:
net.refractions.udig.ui.menuBuilders

Since:
1.1

Description:
Provides a mechanism to configure the menu bar and cool bar without supplying your own application. Use this if you creating your own uDig based application but do not wish to supply your own Application or WorkbenchAdvisor

Configuration Markup:

<!ELEMENT extension (menuBuilder)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT menuBuilder EMPTY>

<!ATTLIST menuBuilder

id    CDATA #REQUIRED

class CDATA #REQUIRED

>

  • id -
  • class -

Examples:

API Information:
The class attribute must be a fully qualified identifier for an implementation of net.refractions.udig.ui.MenuBuilder.

Supplied Implementation:
The default implementation for uDig is provided in plug-in "net.refractions.udig.ui" in the class net.refractions.udig.ui.UDIGMenuBuilder.
