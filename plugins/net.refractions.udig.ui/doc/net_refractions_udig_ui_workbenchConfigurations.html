Workbench Configurations

Identifier:
net.refractions.udig.ui.workbenchConfigurations

Since:
1.1

Description:
Provides a mechanism to configure the workbench window without supplying your own application. Use this if you creating your own uDig based application but do not wish to supply your own Application or WorkbenchAdvisor

Configuration Markup:

<!ELEMENT extension (workbenchConfiguration)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT workbenchConfiguration EMPTY>

<!ATTLIST workbenchConfiguration

id    CDATA #REQUIRED

class CDATA #REQUIRED

>

  • id -
  • class -

Examples:

API Information:
The class attribute must be a fully qualified identifier for an implementation of net.refractions.udig.ui.WorkbenchConfiguration.

Supplied Implementation:
The default implementation for uDig is provided in plug-in "net.refractions.udig.ui" in the class net.refractions.udig.internal.ui.UDIGWorkbenchConfiguration.
