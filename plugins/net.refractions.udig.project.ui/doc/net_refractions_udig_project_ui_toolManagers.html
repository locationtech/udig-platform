toolManagers

Identifier:
net.refractions.udig.project.ui.toolManagers

1.1

Description:
This extension point is used to declare possible candidates for the use the IToolManager used by ApplicationGIS. The one that is actually to be used must be declared in your application's plugin_customization.ini file, using the key specified at IToolManager.P_TOOL_MANAGER.
