Connection Factory

Identifier:
net.refractions.udig.catalog.ui.connectionFactory

Since:
UDIG 1.1

Description:
The extension point that must be implemented when adding new Services to the catalog. The connection factory is used by drag and drop to check if a given input can be used to create a service. In addition it specifies the wizard pages for the import wizard.

Configuration Markup:

<!ELEMENT extension (factory , wizardPage+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>

This extension point provides a factory which is used to provide connection information for a specific service based on context.

<!ATTLIST factory
