>

Service being registered with the uDig catalog


  • class - Implemnetation of ServiceExtension, allowing service to be created from a URL or Map of connection parameters. ServiceExtension2 allows for better error reporting in the event a connection could not be made.
  • id - An id for the service; used in logs and error reporting to indicate the service used.
  • name - A human readable name of the Service
